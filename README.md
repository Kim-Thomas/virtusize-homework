# virtusize-homework
Homework for Virtusize, a login page followed by an edit account page. Made with Vue.js/VueX/Vue Router

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
